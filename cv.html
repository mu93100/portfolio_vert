Pour mettre un fichier PDF dans Visual Studio Code (VSCode) et le publier sur GitHub, voici les étapes simples : 1. Mettre un fichier PDF dans VSCode Très simplement, place ton fichier PDF dans le dossier de ton projet ouvert avec VSCode (par exemple dans un dossier docs/ ou racine). Sur VSCode, tu pourras alors voir ce fichier dans l’explorateur de fichiers. Pour visualiser le PDF dans VSCode, tu peux installer une extension comme vscode-pdf : Clique sur l’icône Extensions (à gauche) Recherche pdf Installe vscode-pdf (ou une autre extension similaire) Ensuite, double-clique sur ton PDF pour l’ouvrir directement dans VSCode avec la visualisation intégrée . 2. Ajouter le PDF dans ton dépôt GitHub Assure-toi que le fichier PDF est dans le dossier de ton projet local (celui que tu commits). Dans le terminal (ou Git intégré à VSCode) : bash git add chemin/vers/ton_fichier.pdf git commit -m "Ajout du fichier PDF" git push origin branche Une fois poussé, tu verras le PDF apparaître dans ton dépôt GitHub, où il sera accessible et téléchargeable par les visiteurs. Exemple simple pour un fichier PDF ou tout autre fichier : xml Télécharger le PDF href="..." indique l’emplacement du fichier (relatif ou absolu). download indique au navigateur de lancer le téléchargement au clic au lieu d’ouvrir le fichier. Tu peux mettre aussi un nom personnalisé comme : Télécharger Important : Le fichier doit être accessible depuis ton serveur ou hébergement web. Sur GitHub, si tu veux un lien direct vers un fichier à télécharger (par exemple un PDF dans un dépôt), utilise l’URL brute (« raw »). Exemple : https://raw.githubusercontent.com/nom-utilisateur/nom-depot/main/chemin/fichier.pdf Puis ajoute ce lien dans ta balise .
